# AVIONA V3 Lite

This package is a lighter, bilingual version of the AVIONA V2.2 prototype.

Intent:
- Keep the updated Web3 aviation positioning.
- Shorten the homepage by approximately 35-45% compared with the full V2.2 version.
- Reduce repetition across subpages.
- Preserve bilingual EN / 中文 toggle behavior.
- Preserve responsive/mobile fixes.

Recommended use:
- Create a separate GitHub repository and separate Vercel project for V3 Lite.
- Do not overwrite the V2.2 prototype unless you decide V3 Lite is preferred.

Files updated:
- index.html
- why-aviona.html
- aircraft.html
- ways-to-participate.html
- about.html

Assets are included in the full package.
